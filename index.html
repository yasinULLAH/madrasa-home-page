About Madrasa Portal

Madrasa Portal is a comprehensive single-page PHP web application that uses SQLite as its backend database. It provides a complete management solution for Islamic schools, enabling them to manage announcements, teachers, courses, timetables, resources, and more - all from a single interface.

The application is built with modern web technologies and follows best practices for performance, security, and user experience. It features a responsive design that works well on all devices, from desktop computers to mobile phones.

User Roles

Public Users

Visitors to the site can:

  • View the homepage with general information about the madrasa
  • Browse announcements and news
  • Check class timetables
  • View teacher profiles
  • Explore available courses
  • View and filter the gallery
  • Download resources
  • Read testimonials and submit their own
  • Read FAQs
  • Submit inquiries through the contact form

Admin Users

Administrators can:

  • Access the admin dashboard with statistics and quick actions
  • Create, edit, and delete announcements
  • Manage teacher profiles
  • Add, update, and remove courses
  • Create and manage class timetables
  • Upload and organize gallery images
  • Manage downloadable resources
  • Approve and manage testimonials
  • Create and organize FAQs
  • View and respond to contact form submissions
  • Update site settings
  • View activity logs
  • Edit profile and change password

Technical Specifications

Backend

  • PHP 7.4+ for server-side logic
  • SQLite database for data storage
  • RESTful API architecture for data operations
  • Single file application for easy deployment

Frontend

  • HTML5, CSS3, and JavaScript
  • Responsive design that works on all devices
  • Single-page application (SPA) architecture
  • No page reloads - content loaded dynamically via AJAX
  • Dark/light mode theme toggle
  • Accessibility features

Security Features

  • Secure password hashing using PHP's password_hash()
  • Session-based authentication
  • Input sanitization to prevent XSS attacks
  • CSRF protection
  • Activity logging for admin actions
  • Secure file upload handling

Installation Instructions

  1. Ensure your server has PHP 7.4+ with SQLite extension enabled
  2. Download the single PHP file (index.php)
  3. Create an 'uploads' directory in the same location and make it writable
  4. Upload the file to your web server
  5. Access the application through your web browser
  6. Log in with default credentials (username: admin, password: admin123)
  7. Change the default password immediately

Server Requirements

  • PHP 7.4 or higher
  • SQLite extension
  • GD or ImageMagick extension (for image processing)
  • Write permissions for the application directory (for SQLite database and uploads)

Usage Guide

Admin Login

To access the admin functionalities:

  1. Click on the "Admin" link in the navigation menu
  2. Enter your username and password
  3. After successful login, you'll be redirected to the admin dashboard

Managing Content

As an admin, you can manage various types of content through their respective pages. The typical workflow is:

  1. Navigate to the section you want to manage (e.g., Announcements, Teachers, Courses)
  2. Use the "Add" button to create new entries
  3. Use the "Edit" button to modify existing entries
  4. Use the "Delete" button to remove entries

Site Settings

To customize the site:

  1. Log in as an admin
  2. Go to the admin dashboard
  3. Click on "Site Settings"
  4. Navigate through the tabs to update various settings
  5. Click "Save Settings" to apply changes

Customization

The application is designed to be easily customizable. Here are some common customizations:

Changing Theme Colors

To modify the color scheme, edit the CSS variables in the :root selector at the beginning of the style section:

:root {
    --main: #2A76D2;    /* Primary color */
    --dark: #151B27;    /* Dark color for footer and dark mode background */
    --light: #F5F7FB;   /* Light background color */
    --card: #fff;       /* Card background color */
    --text: #222;       /* Main text color */
    --accent: #55C594;  /* Accent color for highlights and hover states */
    /* ... */
}

Adding New Features

To add new functionality:

  1. Add appropriate table(s) to the database initialization in the PHP section
  2. Create API endpoints for the new feature
  3. Add UI elements and JavaScript functions for the new feature
  4. Update the navigation menu if needed
